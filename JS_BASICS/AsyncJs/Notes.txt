Most async code will be a part of an API or library

1. XMLHttpRequest and fetch
2. jQuery Ajax, axios, other http libraries
3. Node.js fs(file system) module

Few ways to work with async code

1. Callbacks
2. Promises
3. Async/await

REST APIs
1. GET - retrieve data from a specified resource
2. POST - submit data to be processed to a specified resource
3. PUT - update a specified resource
4. DELETE - delete a specified resource
